# Mobile App

This folder contains the mobile application that works with the Rust backend.

## Features
- User authentication
- Access to account information
- Transaction history
- Institution details

## Getting Started

### Prerequisites
- Node.js
- React Native CLI

### Installation
1. Clone the repository.
2. Navigate to the mobile-app directory.
3. Run `npm install` to install dependencies.

### Running the App
- Use `npm start` to launch the app in development mode.

### Building for Production
- Use `npm run build` to create a production build.
